转专业记录

May 14, 2026 • 4 分钟阅读 • 记录

终于转专业也算是告一段落了，从 CS 转到了 CS ，算是走上了预想的道路？其实一直都是想要转计的，过程还挺漫长。

高考结束后-刚入学

高考确实考得不理想，以至于当时都没有料到能捡到网安（确实也是招生情况很差的省份了），纯纯捡漏，如果没有来着，应该是要到同济读统计类了，会离家近一些。

刚报到那天，学长 zzx 讲话，当时就听说他转了计，而且还拿过很多网安比赛的奖项，这样的佬都去计了，也算是让我有一定想法转计吧。

一直以来都很想打 ACM ，毕竟之前 OI 也没打出什么成绩，但说实话，高考后的暑假比较懒散，很多事情没有先去问清楚，军训期间才去问管竞赛的老师，原来校队的新生选拔早就过了（很多东西确实就限定在计院或者东校区），不过现在看来，实力也不足以选上。
刚开学的时候查 oierdb 在同专业找了几个疑似前 OI 选手问了问，先组了一个队伍，虽然什么正式比赛也没能参加，就参加了校赛和新手赛，分别拿了银牌和金牌（刚好省赛名额外），也感谢学长Lxl的帮助和指导。

大一上

浑浑噩噩过完的，感觉松散得很，一个是因为很多学科都是入门知识，感觉都很简单，而且像大学物理（工），说白了就和初高中物理差不多，超出的部分也在初高中强基学过了，感觉就是高中吃初中老本，大学吃高中老本。感觉没有什么课想上，没有什么东西想学，感觉挺荒废的。
和队友模拟打了几场 acm ，感觉自己好菜啊，打了个校赛（感谢 qjj 带飞），然后就没有下文了。继续浑浑噩噩到了期末，直到考完，才想起来，我想转专业来着，就去找 zzx 学长 uniya 学姐问了问，才知道对我高数成绩略有要求（当时还挺慌的，没想到给分很松），也加了转计群。高数出成绩之后算是下决心要转计了，毕竟网安也不错，肯定不能往差了转可惜了不能享受网安的宿舍。计科应该氛围更卷一些，压力可能也会更大，但是我确实感觉在网安有些太松散了，尽管没有说拿到多高的绩点。

大一下

依旧无所事事，没想着费多大力，而且感觉比上学期更加摆了，不过也许是转计临近了，转计群也越来越热闹。我的目标其实挺明确的，就是复习高数和程设，确保笔试能通过，基本就稳进了。或许是我对自己太过自信，又或许是太轻视其他同学，也有可能是我觉得转计的人也会如去年一般少，竞争不会多激烈，至少在出笔试名单前，我都没有认真复习的想法（实际上出了也没有很认真复习）。

转计名单出的时候，我挺焦虑的，毕竟比去年多了那么多人，报录比超过2，而且我的高数相当不扎实，相当一部分知识点我的印象是模糊的，模糊到无法下笔，也就只有极限和积分能够用一点点固定的套路稍微写两步。我的程设课也几乎没有认真上过，靠着积累的算法竞赛经验去完成C语言基础的考核，对我来说有优势也有很大干扰。即便如此，我也拖到了最后，给自己准备转专业的借口不去上课（甚至借此翘掉了英语课），然后实际也没怎么复习，也就最后一天稍微突击了一下，高数不会的还是一窍不通，程设准备的还是利用部分 C++ 的写法，毕竟当时通知和试机的都用的 g++11 编译器（甚至到考场的试机都是和前一天一样的）。

转计笔试

计的老师真会挑时间，挑了一个暴雨天，而且还得往东校，我一出门就完全湿透，无处落脚，最后还是决定穿上我的拖鞋，准备一件换的衣服，就这样去笔试了，然而我坐完地铁雨就小了。

今年是先进行程设，再考高数，我的预期是程设拿满然后高数及格（然而都没有做到）。
题目参考 机试题目 和 笔试题目 （感谢烛神的网站和大家的资料收集
程设一开题我就懵了，我连 iostream 都用不了，更别提我考前准备的各种 C++ 简洁写法了（这也导致我最后根本就没想过去用 __int128 ），迅速调整到了 C 状态，刚刚打的板子也废了（虽然好像也没用），确实挺搞人心态的。刚开始还悠哉悠哉的，确实前面很简单，我记得好像最后三题剩了一小时还是更久，其实应该做更快些的，倒数第三题数据排序虽说用的 qsort ，还是调了一会，最后剩了40分钟做最后两题，动态数组和比大小，先去搞了下比大小，它的数据范围有点大，我还以为是实数，没有用 __int128 ,就用的 long double 被卡了，事后觉得打个表过一下人造的数据说不定就能继续过随机，但没用了。动态数组一直有点问题，最后好像是没改数组大小，真来不及了，有点可惜，最后86好像。

迅速进入到高数阶段，✍️✍️✍️🤔🤔🤔💤💤💤，唯一有把握的是级数，其他全不会，估分及格。。。的一半。

感觉有点凉透了，一个前 OI 选手，一个网安的学生，竟然连转计科都要失败吗，有点可笑了。
后来听说 zzx 学长也是第一年没过第二年进的，感觉心里有点平衡了（

笔试出分

笔试结束之后就是紧张刺激的 ctf 新手赛，感谢 Akurisu 带飞 orz.
碰巧（计院太能拖了）是在最后一个比赛日中午公布了进面名单，并私法了笔试情况：

好消息是高数又一次改的松，比估分高了20来分，坏消息是高数依旧没及格，但是总共录50个计科，我排32，而且前面还有信计的，感觉已经稳了。

打完 ctf ，开启五一假期，几乎没睡觉，写了两天 wp 并且优化了我的 Code Golf，第三天又和高中同学去玩了一整天，累到虚脱，第四天睡了一天，竟然假期就要结束了， ctf 害人不浅。

面试

面试些许紧张，害怕自己什么都答不出，又是前面开摆，最后一晚突击的多一点。
抽到第3个面试，路上编了点自我介绍，没想到后面真的按我介绍问了点，感觉面试水水的，但是又让我等了好几天才出结果是什么意思？

尾声

三年后毕业时，我是否又会想起那个报名了转计的下午，那个准备着笔试面试的深夜？
